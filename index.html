Cryptocurrencies API

Please use https://cryptocurrencies-api.herokuapp.com/api to access data.

To access a single resourse, make GET request to https://cryptocurrencies-api.herokuapp.com/api/:coin-name, where ':coin-name' is the name of cryptocurency.

For example, GET https://cryptocurrencies-api.herokuapp.com/api/bitcoin will return JSON object with the following properties:

{ "name": "Bitcoin", "ticker": "BTC", "type": "PoW", "max supply": "21,000,000", "whitepaper": "https://bitcoin.org/bitcoin.pdf" }

List of available currencies: Bitcoin, Ethereum, Tether, Cardano, Solana. If the request is unknown, the object with unknown properties will be returned.
